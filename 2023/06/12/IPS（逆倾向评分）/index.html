# IPS（逆倾向评分）

# 1 概述

# 1.1 拟解决的问题

因为选择偏差的存在，推荐系统总是推荐给用户推荐用户喜欢的 Item，不喜欢的 Item 得不到曝光的机会。

基于调整因果推理的模型和估计器，提出了处理选择偏差的原则性方法。

# 1.2 主要贡献

  • 使用倾向加权技术来评估推荐系统的质量
  • 提出经验风险最小化框架，用于在选择偏差下学习推荐系统，并推导了推广误差界
  • 使用经验最小化框架推导出一种矩阵分解方法，该方法可以解释选择偏差，同时保持概念上的简单性和高度可扩展性
  • 探索了在观测环境中估计倾向的方法，其中选择偏差是由用户的自我选择造成的，并且我们描述了框架对错误指定倾向的鲁棒性。

# 2 IPS（ Inverse-Propensity-Scoring）

# 2.1 前提

  • Y 表示真实的评分矩阵
  • P 表示用户对电影评分的概率
  • O 反映有多少用户参与评分
  • Y^1、Y^2\hat Y_1、\hat Y_2Y^1​、Y^2​ 表示评分的预测

IPS 的是设计一个 loss 函数来提高模型的精度，纠正偏差，在整个训练的过程改变的只是 loss 函数，训练的模型可以是 MF、Network 等。

假设能获得真实的，无偏差的预测值Y^\hat YY^，则真实的损失为：

# 2.2.3 实验设置

数据集：ML100K

真实值YYY 是已知的，因此可以用于计算误差，用观察到一个评级Yu,iY_{u,i}Yu,i​ 的概率Pu,iP_{u,i}Pu,i​ 来模拟观察到的边缘概率分布。

预测值皆是模拟出来的，并不是由训练模型得到的，此举只是为了证明该损失函数的优越性

REC_ONES: 预测矩阵Y^\hat YY^ 与YYY 真实的评分矩阵是一样的，作者随机地让一些评分为 1 的变成 5，也就意味着一半的 5 是真实的 5，另一半是评分为 1 的内容

REC_FOURS：将评分为 4 的变成 5，操作与 REC_ONES 一致。

ROTATE: 当真实评分Yu,i≤2Y_{u,i}\leq2Yu,i​≤2 时，则预测评分Y^u,i=Yu,i−1\hat Y_{u,i}=Y_{u,i}-1Y^u,i​=Yu,i​−1。当Yu,i=1Y_{u,i}=1Yu,i​=1 时，Y^u,i=5\hat Y_{u,i}=5Y^u,i​=5

SKEWED: 预测值Y^u,i\hat Y_{u,i}Y^u,i​ 从服从正态分布的N(Y^u,iraw∣u=Yu,i,σ=6−Yu,i2)N({\hat Y^{raw}_{u,i}|u=Y_{u,i},\sigma=\frac{6-Y_{u,i}}{2}})N(Y^u,iraw​∣u=Yu,i​,σ=26−Yu,i​​) 中抽样

COARSENED: 将Yu,i≤3Y_{u,i}\leq3Yu,i​≤3 的置为 3，其余的等于 4

结论：IPS 和 SNIPS 相对于 Naive 方法有更小的偏差，SNIPS 能够降低 MAE 的 IPS 标准偏差，但不能降低 DCG 的 IPS 标准偏差
