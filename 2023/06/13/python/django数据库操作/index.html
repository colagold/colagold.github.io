# django 数据库操作

# 1 ORM

Django 开发操作数据库更简单，内部提供了 ORM 框架。

ORM 可以理解为翻译层，能够将代码转化成 SQL 语句，然后传递给驱动，去操作数据库

# 2 安装第三方模块

最新的 django 一般采用 mysqlclient 作为底层驱动

# 3 ORM 的使用

ORM 可以帮助我们做两件事：

  • 创建、修改、删除数据库中的表（不用你写 SQL 语句）。 【无法创建数据库】
  • 操作表中的数据（不用写 SQL 语句）。

# 3.1 创建数据库

需要自己手动创建数据库，因为 ORM 无法帮助我们创建数据库。
